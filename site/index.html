Gitnotate

Sub-line commenting for GitHub PR reviews

Comment on specific words, sentences, or phrases within Markdown files — not just whole lines. Gitnotate brings precision to your code review workflow.

Install for Chrome Install for Edge View Source

🎯 Precise Selection

Select specific words, phrases, or sentences within Markdown content — go beyond GitHub's line-level commenting.

🔒 Privacy First

All data stays in your browser. No servers, no analytics, no tracking. Gitnotate makes zero network requests.

🪶 Tiny Footprint

Just a small ^gn:42:R:10:25 tag in your comment — hidden by the extension, unobtrusive without it. No extra files, no infrastructure.

💬 Works for Everyone

Comments are standard GitHub PR comments. Reviewers without the extension see them normally — threading, @mentions, and reactions all work.

⚡ Zero Config

Install, enable a repo, and start reviewing. No tokens, no setup, no accounts. Works on any repository with pull requests.

📦 Install

  1. Visit the Gitnotate page on Chrome Web Store
  2. Click Add to Chrome
  3. The Gitnotate icon appears in your toolbar — you're ready!
  1. Visit the Gitnotate page on Edge Add-ons
  2. Click Get to install
  3. The Gitnotate icon appears in your toolbar — you're ready!

Edge can also install extensions from the Chrome Web Store.

  1. Download gitnotate-v0.1.0.zip from the latest release
  2. Extract the zip to a folder on your computer
  3. Open chrome://extensions or edge://extensions
  4. Enable Developer mode (toggle in the top-right)
  5. Click Load unpacked → select the extracted folder
  6. The Gitnotate icon appears in your toolbar — you're ready!

To update: download the new zip, extract to the same folder, then click the reload ↻ button on the extension card.

  1. Clone the repo: git clone https://github.com/pedrofuentes/gitnotate.git
  2. Install dependencies: pnpm install
  3. Build: pnpm build
  4. Open chrome://extensions or edge://extensions
  5. Enable Developer mode
  6. Click Load unpacked → select packages/browser-extension/dist/
